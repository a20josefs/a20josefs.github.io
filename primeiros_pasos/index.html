# Iniciando compoñente

Para mostrar todo este proceso utilizaremos un botón personalizado ao que se lle engadirán propiedades para poder modificar as cores.

O primeiro paso é crear unha clase en java que extenda da clase do compoñente personalizado que nos interesa personalizar e ademais implementar serializable. Neste caso extenderemos de JButton.

[!WARNING] Este exemplo explícase para unha propiedade simple. Se desexamos facer unha propiedade deberemos facer un editor personalizado, pero eso explicarase mais adiante


package com.mycompany.javabeans_exer1;

import java.awt.Color;
import java.io.Serializable;
import javax.swing.JButton;

/**
 *
 * @author a20josefs
 */
public class BotonPersonalizado extends JButton implements Serializable {
    //propiedades que vamos a engadir 
    Color corFondo;
    Color corTexto;
    
    //o constructor non deve ter parametros
    public BotonPersonalizado() {
    }
    
    public Color getCorFondo() {
        return corFondo;
    }
    
    //os seter e os geter permitiran que netbeans visualice (get) e edite (set) as propiedades

    public void setCorFondo(Color corFondo) {
        this.corFondo = corFondo;
        setBackground(corFondo);
    }
    
    public Color getCorTexto() {
        return corTexto;
    }
    
    public void setCorTexto(Color corTexto) {
        this.corTexto = corTexto;
        setForeground(corTexto);
    }
    
}
