# Compoñente con un editor personalizado

A continuación vamos a explicar como facer un compoñente con unha propiedade complexa.

Para isto vamos a necesitar un editor personalizado que nos permita editar esa propiedade.

Para este exemplo vamos a crear un deditor que nos permita editar a cor de fondo e a de texto con unha única propiedade complexa

# Modificando a clase principal

O primeiro paso é modificar a clase principal


public class BotonPersonalizado extends JButton implements Serializable {
    
    ColoresBoton coloresBoton;
    
    public BotonPersonalizado() {
    }
    
    public ColoresBoton getColoresBoton() {
        return coloresBoton;
    }
    
    public void setColoresBoton(ColoresBoton coloresBoton) {
        this.coloresBoton = coloresBoton;
        setBackground(coloresBoton.getCorFondo());
        setForeground(coloresBoton.getCorTexto());
    }
    
}

Tamém deberemos crear uncha clase nova para a propiedade complexa. O tipo desa propiedade será a clase que vamos a crear

public class ColoresBoton {

    Color corFondo;
    Color corTexto;

    public ColoresBoton(Color corFondo, Color corTexto) {
        this.corFondo = corFondo;
        this.corTexto = corTexto;
    }

    public Color getCorFondo() {
        return corFondo;
    }

    public void setCorFondo(Color corFondo) {
        this.corFondo = corFondo;
    }

    public Color getCorTexto() {
        return corTexto;
    }

    public void setCorTexto(Color corTexto) {
        this.corTexto = corTexto;
    }

}

# PropertyEditorSupport

Tamén deberemos crear unha clase que extenda de propertyEditorSupport

public class ColorPropertyEditorSupport extends PropertyEditorSupport {

    private PanelEditorPersonalizado panelPersonalizado = new PanelEditorPersonalizado();

    @Override
    public boolean supportsCustomEditor() {
        return true;
    }

    @Override
    public Component getCustomEditor() {
        return panelPersonalizado;
    }

    @Override
    public String getJavaInitializationString() {

        ColoresBoton coloresBoton = panelPersonalizado.getSelectedValue();

        return "new com.mycompany.javabeans_exer1.ColoresBoton("
                + "new java.awt.Color(" + coloresBoton.getCorFondo().getRed()
                + "," + coloresBoton.getCorFondo().getGreen()
                + "," + coloresBoton.getCorFondo().getBlue()
                + "),"
                + "new java.awt.Color(" + coloresBoton.getCorTexto().getRed()
                + "," + coloresBoton.getCorTexto().getGreen()
                + "," + coloresBoton.getCorTexto().getBlue()
                + "))";

    }

    @Override
    public Object getValue() {
        return panelPersonalizado.getSelectedValue();
    }

}
